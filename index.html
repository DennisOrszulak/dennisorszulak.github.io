Smart Valve App

Swift MVC Auto Layout CoreBluetooth iOS Charts Toast-Swift JSON Git

The Electronics Inc. Smart Valve App connects to EI's shot peening equipment, called MagnaValves, via Bluetooth to give customers control over their next generation valves. They are able to monitor and record real time data, perform their own flow calibrations with multiple presets, export data, and change various settings on the valve.

EE Number Cruncher

Swift MVC Auto Layout Unit Testing Git

This app is a collection of electrical engineering calculators and converters. I wanted to make an app that would be helpful, quick, and easy to calculate common formulas that I used everyday. I included calculators for Ohms law, voltage dividers, parallel/series resistors, parallel/series capacitors, resistor color codes, RC circuits, and decimal/binary/hex/octal conversions. There is also a reference list that provides a quick overview for various electrical components. Check it out on the app store!
